根据GB/T31831-2015规范，在频率为9Hz至3125Hz时，波动深度高于f*0.08/2.5%，就可能会对人体造成影响。\
而LCD显示器的LED背光的波动率一般都为100%，\
所以，就算在全黑环境中使用PWM调光的LCD显示器，只要显示器LED背光的PWM频率超过3125Hz，频闪就不会对身体造成任何影响，不用担心伤眼问题。\
此结论同样适用于家用照明设备，比如荧光灯、LED顶灯、台灯等设备的频闪安全性界定。



## 小小的冷知识
由于硬件晶振的物理限制，显示的频率是目标值，实际生效频率可能会有 5% 以内的物理偏差，属正常现象。\
举个例子: Surface Go 2 软件显示默认频率是 1000Hz, 但是示波器硬件测量实际是 962Hz
